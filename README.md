# Predictive Student Risk Analysis & Early Warning System

## Project Overview
This project predicts student risk levels (High, Medium, Low) using Machine Learning models.

## Dataset
Student Performance Dataset

## Models Used
- Random Forest
- Gradient Boosting
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Visualizations
- Risk Distribution
- Correlation Heatmap
- Feature Importance
- Model Comparison

## Business Impact
Helps educational institutions identify at-risk students early and provide timely interventions.
